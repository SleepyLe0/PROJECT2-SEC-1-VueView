# What is turn-based game?
เกมแบบ Turn-Based ที่มีการโจมตีกัน เป็นแนวเกมที่ผู้เล่นจะทำการเล่นเป็นรอบๆ (Turn) ตามลำดับ  โดยจะมีการโจมตีสลับกันระหว่างผู้เล่นและ
ฝ่ายตรงข้าม การเล่น Turn-Based จะต้องวางแผนให้ดีในการเล่นแต่ละ Turn เพื่อเอาชนะฝ่ายตรงข้าม  

กติกาเกม คือ เกมจะเล่นเป็น Turn โดยจะแบ่งเป็น Turnของผู้เล่น และ Turnของศัตรู ซึ่งตอนเริ่มเกมทั้งสองฝั่งจะได้รับ  skill point 
1 point โดย skill point สูงสุดที่จะได้รับตอนเริ่ม turn จะเพิ่มขึ้นตามจำนวนTurn ที่เล่น สูงสุด 4 point และ Skill point ที่ได้รับมาจะสามารถนำมาใช้ Action ในแต่ละ turn และในTurn แรก คนที่ออก Action ทีหลังจะได้รับโบนัส Skill Point เพิ่ม 1 แต้ม

# How to play? 
ในแต่ละ  Turn ของผู้เล่นสามารถใช้ได้ 3 ปุ่ม โดยจะมี Attack Button, Defense Button, และ Charge Button โดยการกดปุ่มหนึ่งครั้ง จะใช้ Skill Point 1 Point ซึ่งแต่ละ Action มีดังนี้
1. Attack Button: จะเพิ่มจำนวน Attack ของผู้เล่น ซึ่งการ Attack จะแสดงผลหลังจากจบ Turn ผู้เล่น เมื่อจบ Turn ของผู้เล่น จะนำ Attack Point  ของผู้เล่น ไปคูณกับ ATK ของตัวละครเพื่อสร้างความเสียหายต่อศัตรู
2. Defense Button: จะเพิ่มจำนวน Defense ของผู้เล่น ซึ่งการ Defense จะแสดงผลหลังจากศัตรู Action เสร็จ เมื่อจบ Turn ของศัตรูจะนำ Defense Point ของผู้เล่นไปลบ Attack Point ของศัตรู
3. Charge Button: จะเพิ่มจำนวน Charge  ของผู้เล่น ซึ่งการ Charge  จะแสดงผลเมื่อเริ่ม Turn ถัดไปของผู้เล่น ในการ Charge แต่ละครั้งจะเป็นการเพิ่ม Skill Point ใน Turn ถัดไปของผู้เล่น Charge Point มีได้มากสุดแค่ 4 ครั้งต่อ Turn 
4. ในแต่ละ Turn เราจะต้องใช้ Skill Point ให้ครบก่อนถึงจะจบ Turn ได้ หากฝ่ายใดเลือดหมดก่อนก็จะถือเป็นฝ่ายแพ้ ดังนั้นผู้เล่นจะต้องใช้กลยุทธ์อย่างชาญฉลาดในการเลือกใช้ปุ่มเพื่อคว้าชัยชนะในเกมนี้
